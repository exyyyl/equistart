# Equicord Unified Launcher

Ультимативное решение для автоматического запуска Discord с установленным Equicord в один клик.

## 🚀 Как пользоваться
1. Скачайте/скопируйте файл `EquiLauncher.bat` в удобное место.
2. Просто запустите его двойным кликом.
   - Скрипт сам закроет Discord (если он открыт).
   - Очистит старые патчи (чтобы избежать ошибок).
   - Установит/обновит Equicord.
   - Запустит Discord.

## ⚙️ Автозагрузка
Скрипт поддерживает автоматическую установку в автозагрузку Windows.

### Способ 1: Автоматический (через скрипт)
1. Откройте терминал (PowerShell или CMD) в папке со скриптом.
2. Запустите команду:
   ```cmd
   EquiLauncher.bat --startup
   ```
   *Это создаст ярлык в папке автозагрузки, который будет запускать скрипт в свернутом режиме при включении ПК.*

### Способ 2: Ручной
1. Нажмите `Win + R`, введите `shell:startup` и нажмите Enter.
2. Создайте ярлык для файла `EquiLauncher.bat` в открывшейся папке.
3. (Опционально) В свойствах ярлыка выберите "Окно: Свернутое в значок", чтобы окно не мелькало при запуске.

## 🛠 Особенности
- **Все в одном**: Никаких сторонних `.ps1` файлов, всё внутри одного `.bat`.
- **Чистая установка**: Перед каждым запуском скрипт восстанавливает `index.js`, что гарантирует отсутствие ошибок "Discord is corrupted".
- **Тихий режим**: Большинство операций происходит в фоновом режиме.

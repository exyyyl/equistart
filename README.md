# 🚀 EquiLauncher
1. Скачайте/скопируйте файл `EquiLauncher.bat` (и опционально `launcher.ps1`) в удобное место.
2. Просто запустите `EquiLauncher.bat` двойным кликом. Откроется интерактивное меню, где вы сможете выбрать:
   - Установку и запуск Equicord или Vencord.
   - Запуск в режиме отладки (с созданием лога ошибок).
   - Добавление выбранного мода в автозагрузку Windows.

Скрипт автоматически:
- Закроет Discord (если он открыт).
- Скачает нужный установщик, если его нет.
- Пропатчит Discord нужным модом (Equicord или Vencord).
- Запустит Discord.

## ⚙️ Автозагрузка
Скрипт поддерживает автоматическую установку в автозагрузку Windows для любого из модов.

### Способ 1: Интерактивный (Рекомендуемый)
1. Запустите `EquiLauncher.bat` двойным кликом.
2. В появившемся меню выберите пункт добавления в автозагрузку (пункты 5 или 6).

### Способ 2: Через командную строку
1. Откройте терминал в папке со скриптом.
2. Для Equicord:
   ```cmd
   EquiLauncher.bat --silent
   ```
3. Для Vencord:
   ```cmd
   EquiLauncher.bat --silent-vencord
   ```

### Способ 3: Ручной
1. Нажмите `Win + R`, введите `shell:startup` и нажмите Enter.
2. Создайте ярлык для файла `EquiLauncher.bat` в открывшейся папке.
3. В свойствах ярлыка допишите нужный аргумент (`--silent` для Equicord или `--silent-vencord` для Vencord).
4. (Опционально) В свойствах ярлыка выберите "Окно: Свернутое в значок".

## 🛠 Особенности
- **Поддержка двух модов**: Выбирайте между Equicord и Vencord прямо в меню.
- **Интерактивное меню**: Удобный интерфейс без необходимости вручную прописывать параметры.
- **Режим отладки**: Позволяет найти причину ошибки, сохраняя детальный лог запуска.
